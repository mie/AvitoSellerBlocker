# Avito seller blocker - userscript для блокировки нежелательных продавцов на Avito

Avito не предоставляет встроенных средств блокировки/скрытия объявлений определенных аккаунтов. Для исправления ситуации был создан скрипт, который позволяет создать свой собственный блок-лист таких продавцов. Для работы скрипта необходимо предварительн установить расширение Tampermonkey.
