# Avito seller blocker - userscript для блокировки нежелательных продавцов на Avito

Avito не предоставляет встроенных средств блокировки/скрытия объявлений определенных аккаунтов. Для исправления ситуации был создан скрипт, который позволяет создать свой собственный блок-лист таких продавцов. Для работы скрипта необходимо предварительно установить расширение Tampermonkey ([Firefox](https://addons.mozilla.org/en-US/android/addon/tampermonkey/), [Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en)).

При включенном расширении Tampermonkey установку можно провести отсюда https://openuserjs.org/scripts/siomi/Avito_Seller_Blocker
